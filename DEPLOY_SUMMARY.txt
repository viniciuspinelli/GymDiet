🚀 COMO USAR AGORA
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  TESTES LOCAIS
   
   $ make install          # Instalar dependências
   $ make dev              # Iniciar com Docker
   $ make migrate          # Executar migrações
   $ make seed             # Carregar dados de exemplo
   
   Acesse: http://localhost:3000
   Login: admin / admin123


2️⃣  FAZER PUSH PARA GITHUB

   $ git add .
   $ git commit -m "GymDiet - Pronto para Render"
   $ git push origin main


3️⃣  DEPLOY NO RENDER (SIGA DEPLOY_CHECKLIST.md)

   • Crie PostgreSQL database no Render
   • Crie Web Service Docker
   • Configure DATABASE_URL
   • Aguarde deploy completar
   • Execute: npm run db:migrate
   • Execute: npm run db:seed
